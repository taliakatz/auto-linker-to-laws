This is my final work as part of the 'Digital Humanities' course.
My purpose is to search after the israeli laws references all over the hebrew written sites,
to 'tag' them and to create a link to an official source which present the law in it's full detailed version.
The link leads to the Isreali Knesset site (which for most of the laws contain a list of some
versions of the law).

The code introduce an automatically linker, that 'cach' all the possible versions for the laws names. 
Different versions for the names occurs when a hebrew word contains some vowels (אותיות אהו"י) or
prepositions (which in Hebrew some of the prepositions are letters)

The way I tried to figure out this issue is by using a regualr expression, which is built according to the
word specifically at runtime. The purpose of that idea is is to make the code independent from any early editing 
of the laws names. So, it is also independent from a permanent database. Finally the linker is caching all the word 
which are in some 'editing distance' from the original name of the law.
